# Usage


## Run

Test the app runs.

```console
$ make run
go run main.go
Hello, World!
go run main.go --name Gopher
Hello, Gopher!
```

Or install it globally as per [Quickstart](quickstart.md) or [Installation](installation.md) and then run it anywhere by name.

```console
$ go-project-template
Hello, World!
```
